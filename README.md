# Event-organize-Contract

A Solidity smart contract for organizing events and selling tickets on the Ethereum blockchain.

## Features
- Create events with name, date, price, and ticket count
- Buy tickets using Ether
- Track remaining tickets
- Transfer tickets between users

## Tech Stack
- Solidity
- Remix IDE
- Ethereum (EVM compatible)

## How to Use
1. Open the contract in Remix IDE
2. Compile using Solidity compiler
3. Deploy on Remix VM or Testnet
4. Interact with functions like `CreateEvent` and `BuyTicket`
